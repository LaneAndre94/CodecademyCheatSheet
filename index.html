CSS Box Model

Properties

Property Name Description
Height and Width The height and width of a content area can be set in pixels or percentages.
Padding Padding is the space between the content area and the border. It can be set in pixels or percent.
Borders Borders surround the content area and padding of an element. The color, style, and thickness of a border can be set with CSS properties.
Margins Margin is the amount of spacing outside of an element's border.
